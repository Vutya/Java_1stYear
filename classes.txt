Объектно-ориентированное программирование

Объекты - это существитеьльные из предметной области.
В программе поддержки обучения будут объекты:
	- студент
	- преподаватель
	- предмет
	- экзамен
	- оценка
	- ...

Объект содержит 1) данные и 2) поведение.

Данные:
1) студент: фио, курс, информация об оценках
2) преподаватель: фио, зарплата, предметы, которые ведёт
3) оценка: значение дата
...

Поведение:
1) студент может получить оценку за предмет
2) студент может распечатать список всеъ своих оценок
3) преподаватель может получить новый предмет для обучения
...

В теории всё так, как написано выше. На практике, в реальных программах примеры
объектов значительно более абстрактны.

String - строки
Integer - целые числа
File - путь на диске
Set - множество объектов
Scanner - устройство, которое читает текст и выдаёт слов из него

DBConnection - связь с БД
CrawlerConfigFactory - фабрика конфигурации кроулера...
Мы будем программировать класс "рациональное число" Rational.
3/5, 8/7, 2 = 2/1, -5 = -5/1
1/3 + 1/6 = 1/2
Класс Rational содержит данные: числитель, знаменатель.
Действия для класса Rational: арифметические действия.

Как оформлять в Java. Узнаем в программе.
