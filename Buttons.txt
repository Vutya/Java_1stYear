ВАЖНЫЕ КНОПКИ В IDEA

Ctrl + Space - автодополнение, если окошко само не отображается
Ctrl + Shift + Space более умное
Enter - выбор из списка предложенных дополнений ИЛИ Tab, который перезаписывает
Лампочка ( Alt + Enter ) слева от кода предлагает исправления
вызов помощи Ctrl + Q
Ctrl + P внутри списка аргументов показывает набор аргументов функции
Изменение имени Shift + F6 или нажать пкм и искать рефакторинг там
Ctrl + D - дублировать строку
выделение переменной Ctrl + Alt + V
Ctrl + Q - помощь по методу
Alt + Ins - по-быстрому написать какой-нибудь get, set, конструктор итд
ещё есть рефакторинг extract method (Ctrl + Alt + M)
